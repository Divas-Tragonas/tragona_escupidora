⚙ Configuración

API Key de Groq (gratuita)

La necesitas para transcribir el audio desde el navegador. Crea una en console.groq.com — es gratuita y tarda 1 minuto.

Se guarda solo en este navegador (localStorage). Solo se envía a Groq al transcribir.

Instalación única

Solo necesitas esto una vez. Después, cada sesión son ~3 clics.

1
yt-dlp — descarga VODs de Twitch
pip install yt-dlp
O descarga el ejecutable en github.com/yt-dlp
2
ffmpeg — convierte el audio
sudo apt install ffmpeg
Windows: winget install ffmpeg  ·  macOS: brew install ffmpeg
3
Claude.ai Business — ya la tienes, no necesitas configurar nada más.

Sobre los datos

Los recaps se guardan en el localStorage de este navegador — sin base de datos, sin servidor, sin coste. Para hacer una copia de seguridad, abre cualquier recap y usa el botón Copiar Markdown.

Coste por sesión: ~$0.05 (solo Groq Whisper). Claude corre sobre tu suscripción Business existente.
